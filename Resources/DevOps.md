# 🚀 30 Days DevOps Challenge : By WebXWizard ✅

Welcome to the **30 Days DevOps Challenge** — a hands-on, structured learning journey to master essential DevOps tools, practices, and workflows in just **30 days**.

This challenge is designed for beginners and intermediate learners who want to **build a strong foundation in DevOps** by learning one concept at a time with practical examples, tasks, and mini-projects.

---

## 📅 Challenge Overview

| Day | Topic | Tools/Concepts Covered |
|-----|--------|------------------------|
| 1-3 | Introduction to DevOps | What is DevOps, SDLC, CI/CD basics |
| 4-6 | Linux Fundamentals | Shell, Commands, File System |
| 7-9 | Git & GitHub | Version control, branching, PRs |
| 10-12 | Docker | Containers, Images, Dockerfiles |
| 13-15 | Kubernetes | Pods, Deployments, Services |
| 16-18 | Jenkins | CI/CD pipelines |
| 19-21 | AWS Basics | EC2, S3, IAM, Cloud concepts |
| 22-24 | Terraform | Infrastructure as Code (IaC) |
| 25-27 | Ansible | Configuration Management |
| 28-30 | Final Project | Complete DevOps pipeline setup |

---

## 🧠 What You’ll Learn

✅ Real-world understanding of DevOps practices  
✅ Hands-on with popular tools like **Git, Docker, Kubernetes, Jenkins, AWS, Terraform, and Ansible**  
✅ How to automate, build, and deploy applications  
✅ Cloud and Infrastructure-as-Code fundamentals  
✅ CI/CD workflow setup from scratch

## 🚀 How to Use This Repo

1. Clone this repository  
   ```bash
   git clone https://github.com/WebXWizard/30-Days-DevOps-Challenge.git
   ```
2. Navigate into the folder  
   ```bash
   cd 30-Days-DevOps-Challenge
   ```
3. Start with **Day 1**, follow the guide and complete daily tasks.  
4. Document your progress and share it on GitHub or LinkedIn!
