## 🚀 How to Use

You can read the `.md` files for theoretical concepts and run the `.sh` files to see the code in action.

To run a script, open your terminal and use:

```bash
bash 02-Variables.sh
```

Or make it executable and run it directly:

```bash
chmod +x 02-Variables.sh
./02-Variables.sh
```

Happy Scripting!
