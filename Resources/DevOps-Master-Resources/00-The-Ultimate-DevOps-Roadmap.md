# The Ultimate DevOps Roadmap

This is your map. You are currently navigating the "Cloud & Automation" phase.

## 1. Fundamentals (The Bedrock)
- [x] **Linux**: File system, Permissions, Bash Scripting.
- [x] **Networking**: DNS, IP Addresses, Ports (HTTP 80, SSH 22).
- [x] **Version Control**: Git & GitHub.

## 2. Containerization (The Package)
- [x] **Docker**: Images, Containers, Dockerfile.
- [ ] **Docker Compose**: Multi-container applications.

## 3. Cloud Computing (The Infrastructure)
- [ ] **AWS / Azure / GCP** (Pick ONE).
- [x] **IAM**: Security.
- [x] **EC2**: Virtual Servers.
- [x] **VPC**: Networking.
- [x] **S3**: Storage.

## 4. Infrastructure as Code (The Blueprint)
- [x] **Terraform**: Provisioning resources via code.
- [x] **Ansible**: Configuration Management (Setting up servers).

## 5. CI/CD (The Factory)
- [x] **Jenkins**: Classic automation server.
- [ ] **GitHub Actions**: Modern cloud-native CI/CD.

## 6. Container Orchestration (The Fleet Commander)
- [x] **Kubernetes**: Deploying and scaling containers.
- [ ] **Helm**: Package manager for K8s.

## 7. Monitoring & Logging (The Eyes)
- [ ] **Prometheus**: Metrics (CPU, RAM).
- [ ] **Grafana**: Visualization (Dashboards).
- [ ] **ELK Stack**: Logging (Elasticsearch, Logstash, Kibana).

---
> **Tip**: Don't try to learn everything at once. Focus on building ONE project that uses all these tools together.
