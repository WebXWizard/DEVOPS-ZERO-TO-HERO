# AWS For DevOps - Beginner to Professional

Welcome to your AWS DevOps journey. This folder contains structured guides to master the most critical AWS services used by DevOps Engineers.

## How to use this folder
Each directory contains a "Mastery" guide for a specific service.

## Curriculum
1.  **[IAM (Identity and Access Management)](./01-IAM/IAM-Mastery.md)**
    - The foundation of AWS security. Learn Users, Roles, and Policies.
2.  **[EC2 (Elastic Compute Cloud)](./02-EC2/EC2-Mastery.md)**
    - Virtual servers in the cloud. Learn instances, security groups, and storage.
3.  **[S3 (Simple Storage Service)](./03-S3/S3-Mastery.md)**
    - Infinite storage. Learn buckets, versioning, and hosting websites.
4.  **[VPC (Virtual Private Cloud)](./04-VPC/VPC-Mastery.md)**
    - Networking. The hardest but most important part. Subnets, Route Tables, Gateways.

## Pre-requisites
- An AWS Free Tier Account.
- Basic understanding of CLI/Terminal.
