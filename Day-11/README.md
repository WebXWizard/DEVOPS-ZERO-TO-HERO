# Day 11 - Docker Networking and Volumes

This folder is a beginner-friendly guide for two important Docker topics:

1. Docker Networking
2. Docker Volumes

By the end of this day, you should understand how containers talk to each other and how containers keep data safe even after they are removed.

## Learning Path

Read these files in order:

```text
Day-11/
|-- 01-Docker-Networking-Beginner-Guide.md
|-- 02-Docker-Volumes-Beginner-Guide.md
|-- 03-Commands-Cheat-Sheet.md
`-- Project-Task-Tracker/
```

## What You Will Learn

- What Docker networks are
- Why containers need networks
- Difference between default bridge and custom bridge network
- How one container can talk to another container by name
- What Docker volumes are
- Why container data is temporary by default
- Difference between named volumes and bind mounts
- How to build a small project using networking and volumes together
